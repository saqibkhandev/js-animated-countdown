# JS Animated Countdown ⏱️

An animated countdown built with HTML, CSS, and JavaScript. This project demonstrates timers and CSS animations for a clean, beginner-friendly countdown component.

## Features
- Animated countdown timer
- Simple, readable code for learning
- No build tools required

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
```
index.html — demo page
styles.css — styling and animations
script.js — countdown logic
```

## Installation / How to run 🚀
1. Clone or download the repo.
2. Open `index.html` in a modern browser.

## Usage
- Start the page to see the countdown animation. Modify CSS or JS to experiment.
